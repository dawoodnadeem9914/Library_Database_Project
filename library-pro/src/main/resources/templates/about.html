Library Management System

An enterprise-level Library Management System developed for CSC3402 / CCS3402 — Database Application Development. The system manages the complete borrowing lifecycle — books, members, loans, fines and reporting — on top of an Oracle database using the Spring MVC architecture.

MVC Architecture

View

Thymeleaf templates + Bootstrap 5 render the UI and submit forms to controllers.

Controller + Service

Controllers map URLs; services enforce business rules such as loan limits and availability updates.

Model

JPA entities map to the five Oracle tables; repositories handle all SQL via Spring Data.
